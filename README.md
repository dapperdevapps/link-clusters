# Internal Link Clusters

WordPress plugin for managing internal-link hub grids (clusters) with admin UI, styling controls, layout modes, and auto-insert functionality.

## Features

- Create and manage link clusters
- Auto-insert clusters into content
- Customizable styling with color pickers
- Font Awesome icon support
- SEO-friendly with Schema.org markup
- Per-link rel attributes (nofollow, sponsored, UGC)
- Custom CSS classes
- Responsive grid layout

## Installation

1. Upload the plugin files to `/wp-content/plugins/internal-link-clusters/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Internal Link Clusters' in the admin menu

## Usage

### Shortcodes

- `[rc_cluster name="cluster-slug"]` - Display a specific cluster
- `[rc_cluster_auto]` - Automatically display cluster for current page

### Settings

Configure clusters, styling, SEO, and icon settings in WordPress Admin → Internal Link Clusters → Settings

## Requirements

- WordPress 5.0 or higher
- PHP 7.2 or higher
